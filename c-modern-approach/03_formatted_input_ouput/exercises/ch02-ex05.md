# 5. Suppose that we call scanf as follows:
```c
scanf("%f%d%f", &x, &i, &y);
```
### If the user enters
#### Input
```bash
12.3 45.6 789
```

### What will be the values of x, i, and y after the call? (Assume that x and y are float variables and i is an int variable.)

```c
#include <stdio.h>

int main(void)
{
  int i;
  float x, y;

  scanf("%f%d%f", &x, &i, &y);

  printf("x: %.1f, i: %d, y: %f\n", x, i, y);

  return 0;
}
```

### Output
```bash
x: 12.3, i: 45, y: 0.6
```
